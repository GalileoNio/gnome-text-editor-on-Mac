Porting notes

移植状态怎么读

Running

应用已经能在 macOS 上启动，并完成了基本 app bundle 包装。

Packaging

应用可以编译，但还在处理图标、schema、启动脚本或发布包结构。

Research

依赖链、平台 API 或 UI 行为仍在调查，暂未进入可运行阶段。

GitHub Pages

无需构建，直接发布静态目录

站点文件都在 site/。仓库内已经包含 GitHub Actions workflow， push 到 main 或手动运行 workflow 后，会把这个目录发布到 GitHub Pages。 

python3 -m http.server 4173 --directory site
